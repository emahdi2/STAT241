<h1>STAT 241 – Biostatistics</h1>
<p><strong>Course Repository for STAT 241: Biostatistics</strong></p>
<p>This repository contains course materials, lecture notes, datasets, software guides, and resources for statistical methods applied to medical and health sciences.</p>

<hr>

<h2>Course Description</h2>
<p>STAT 241 introduces statistical techniques used in medical research and public health. Topics include:</p>
<ul>
    <li>Methods of sampling in medical studies</li>
    <li>Summarizing and presenting medical data</li>
    <li>Demographic statistics</li>
    <li>Survival analysis</li>
    <li>Cross-tabulation analysis</li>
    <li>Inference for means and proportions</li>
    <li>Parametric and non-parametric methods</li>
    <li>Regression models:
        <ul>
            <li>Multiple Linear Regression</li>
            <li>Logistic Regression</li>
            <li>Poisson Regression</li>
            <li>Cox Regression</li>
        </ul>
    </li>
    <li>Sample size determination</li>
</ul>

<p><strong>Software used:</strong> R, MINITAB, SPSS, Microsoft Excel</p>

<hr>

<h2>Learning Objectives</h2>
<ul>
    <li>Summarize and visualize medical data</li>
    <li>Interpret rates, risks, and odds</li>
    <li>Perform statistical inference for means and proportions</li>
    <li>Apply nonparametric methods when assumptions are violated</li>
    <li>Build and interpret regression models</li>
    <li>Analyze survival data using Kaplan–Meier methods</li>
    <li>Design studies and determine appropriate sample sizes</li>
</ul>
